📊 Excel Store Annual Report Dashboard

This project presents a comprehensive store annual report dashboard built using Microsoft Excel. It analyzes sales performance, customer segmentation, order status, and regional insights to support data-driven decision making.

🛠 Tools Used

  • Microsoft Excel
  • Pivot Tables
  • Data Visualization
  • Dashboard Design

📊 Key Insights

  • Monthly sales vs orders trend analysis
  • Customer segmentation (Men vs Women)
  • Order status distribution
  • Top performing states
  • Channel-wise sales contribution
